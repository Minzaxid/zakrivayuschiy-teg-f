# Закрывающий тег

Учебный проект по адаптивной вёрстке из курса **Яндекс Практикум**. 
Это атмосферная страница-дневник в стиле ретро-интерфейсов и пиксель-арта, рассказывающая о пути студента на курсе веб-разработки (от нулевого спринта до финального проекта). Главная техническая особенность проекта — создание сложных микроинтеракций, многослойных анимаций SVG и работа с графическими эффектами.

Логика работы кнопок лайка уже была реализована в готовом JavaScript-файле, я работал над HTML-структурой, подключением шрифтов и написанием продвинутых CSS-стилей и анимаций.

## Что умеет страница

- Адаптируется под разные размеры экранов (десктопные мониторы и мобильные устройства).
- Оживает при взаимодействии: многослойная SVG-иконка сердца плавно закрашивается при наведении, а при клике — пульсирует с эффектом разлетающихся искр.
- Поддерживает нестандартные эффекты наведения на кнопки: фон плавно выезжает, а цвет текста и иконки инвертируется.
- Открывает нативное модальное окно поверх страницы при нажатии на кнопку «Сохранить на память».
- Применяет уникальные визуальные фильтры к каждому изображению карточки.

## Технологии

- HTML5, семантическая разметка (включая работу с нативным тегом `<dialog>`).
- Оптимизация SVG-графики (использование `<symbol>` и `<use>`, подготовка многослойных изображений для стилизации).
- CSS3 (широкое использование CSS-переменных, Flexbox-вёрстка).
- Продвинутая работа с CSS-анимациями (`@keyframes`) и тонкими настройками переходов (`transition` с микро-задержками).
- Адаптивная типографика с использованием математической функции `clamp()`.
- Использование графических фильтров CSS (`filter: grayscale, sepia, invert, hue-rotate` и др.).
- Работа с режимами наложения цветов (`mix-blend-mode: difference`).
- Создание сложных составных фонов (`repeating-linear-gradient`).
- Подключение вариативных и кастомных шрифтов (Inter-Variable и пиксельного PressStart2P).
